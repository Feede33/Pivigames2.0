## 🎨 Paleta de Colores

### Tarjeta de Precio
```css
/* Fondo */
background: linear-gradient(to bottom right, 
  rgba(20, 83, 45, 0.3),   /* Verde oscuro con transparencia */
  rgba(22, 101, 52, 0.2)   /* Verde medio con transparencia */
);

/* Borde */
border: 1px solid rgba(34, 197, 94, 0.5);  /* Verde brillante semi-transparente */

/* Badge de país */
background: rgba(20, 83, 45, 0.4);  /* Verde oscuro */
color: rgb(74, 222, 128);           /* Verde claro */

/* Badge de descuento */
background: rgb(22, 163, 74);  /* Verde brillante */
color: white;

/* Precio final */
color: rgb(74, 222, 128);  /* Verde claro brillante */

/* Precio original (tachado) */
color: rgb(156, 163, 175);  /* Gris */
text-decoration: line-through;

/* Texto de moneda */
color: rgb(156, 163, 175);  /* Gris claro */
```

## 🎯 Casos de Uso Visuales

### Caso 1: Usuario de Argentina ve un juego con descuento
```
1. Abre el modal
   ↓
2. Ve su país: 📍 Argentina
   ↓
3. Ve el descuento: -75%
   ↓
4. Ve el precio original tachado: $3,999.99
   ↓
5. Ve el precio final: $999.99 ARS
   ↓
6. Sabe que es en pesos argentinos
```

### Caso 2: Usuario de USA ve un juego gratis
```
1. Abre el modal
   ↓
2. Ve su país: 📍 United States
   ↓
3. Ve "GRATIS" en grande
   ↓
4. Sabe que puede descargarlo sin costo
```

### Caso 3: Usuario con VPN ve precios de otro país
```
1. Conecta VPN a Japón
   ↓
2. Abre el modal
   ↓
3. Ve: 📍 Japan
   ↓
4. Ve precio en yenes: ¥ 6,000 JPY
   ↓
5. Los precios son de la región japonesa
```
